# 📝 Todo App (React)

A simple and responsive Todo App built with React.js.

## 🚀 About the Project

This project is a basic Todo application created as part of my React learning journey. It helps practice core React concepts such as components, state management, and user interactions.

## ✨ Features

* Add new tasks
* Mark tasks as completed
* Delete tasks
* Simple and clean UI
* Responsive design

## 🛠️ Built With

* React.js
* JavaScript (ES6+)
* HTML5
* CSS3
